# Soma

A remote control for your Samsung TV that runs on your Mac (and soon your iPhone).

It talks to the TV directly over your local network using Samsung's WebSocket
remote protocol — no cloud, no third-party service, no account required.
The TV shows an authorization prompt on first connection, and from then on
the app can send remote-control commands to it.

## Status

- **Mac** — in development. Currently supports volume up/down and mute.
  This is where the project starts.
- **iOS** — planned as the next platform, sharing the same connection logic.

> Your Mac/iPhone and the TV must be on the same local network.
